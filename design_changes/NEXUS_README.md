# Nexus - The MBA Co-Pilot

## Overview
Nexus is an agentic executive assistant prototype designed for MBA students, optimizing for the "Triple Bottom Line": **Academic Excellence**, **Professional Networking**, and **Personal Well-being**.

## Technical Stack

- **Framework**: React 18 with TypeScript
- **Styling**: Tailwind CSS v4
- **UI Components**: Radix UI primitives
- **Charts**: Recharts
- **Animations**: Motion (Framer Motion)
- **Icons**: Lucide React
- **Notifications**: Sonner

## Design Principles

1. **High Information Density, Low Noise**: Monday.com-style grid meets clean timeline views
2. **Agentic UI**: Proactive suggestions with clear reasoning
3. **Biometric Awareness**: Health-conscious scheduling
4. **Context Preservation**: Minimal context switching between academic, professional, and personal domains
5. **Professional Aesthetic**: Business-oriented language and visual design

## Mock Integrations

This prototype demonstrates the UI/UX for:
